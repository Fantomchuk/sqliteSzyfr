    LocalLogs

    Відповідає за додання і створення локальних файлів з комунікатами логів

    Constructor

    new LocalLogs()

    Створюється один екземпляр класу, на якому і проводимо всі операції. Цей екземпляр експортуємо з цього файлу, і будемо його підключати в інших файлах.

    Example
    const LocalLogs = require("./backend/core/LocalLogs");

    Methods

    add(controller, action, msg, jsonopt) → {undefined}

    Створення і додання до файлу нового логу

    Name Type Attributes Description
    controller string

    назва контроллера або модуля в якому викликана функція
    action string

    назва акції яку виконуємо
    msg string

    повідомлення яке хочемо записати
    json object | string | number <optional>
