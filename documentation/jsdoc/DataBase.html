    DataBase

    Відповідає за локальну базу даних

    Constructor

    new DataBase()

    Створюється один екземпляр класу, на якому і проводимо всі операції. Цей екземпляр експортуємо з цього файлу, і будемо його підключати в інших файлах.

    See:
    • https://github.com/mapbox/node-sqlite3/wiki/API
    Example
    const DataBase = require("./backend/core/DataBase");

    Methods

    (async) all(command, params) → {Promise.<object>}

    Використовуємо для побрання невеликих кількостей рекордів(даних). Всі рекорди побераються і тримаються в памяті

    Parameters:
    Name Type Description
    command string

    SQL запит
    params Array

    Параметри до SQL запиту
    Returns:
    Type:
    Promise.<object>
    Example
    const DataBase = require("./backend/core/DataBase");
await DataBase.all('SELECT * FROM users WHERE name=?', ['Nazar]);

    (async) getAllRows() → {Promise.<object>}

    Повертає всі записи з бази даних для табличкі 'Users' Дана функція прикладова, для отримання всіх записів якої небудь табличкі варто скористатися з класи яка називається так само як табличка в базі

    Returns:
    Type:
    Promise.<object>
    Example
    const list = await DataBase.getAllRows();
